プライバシー ポリシー

最終更新日:2024 年 1 月

はじめに

このプライバシー ポリシーでは、お客様が当社の Web サイトにアクセスした際に、当社がお客様の個人情報をどのように収集、使用、保護するかについて説明します。

当社はお客様のプライバシーを保護し、お客様の個人情報が安全かつ責任ある方法で取り扱われるよう努めます。

私たちが収集する情報

あなたが直接提供する情報:

  • お問い合わせフォームを使用する場合の連絡先情報 (名前、電子メール)
  • あなたが提供することを選択したその他の情報

自動的に収集される情報:

  • ブラウザの種類とバージョン
  • オペレーティング システム
  • 訪問したページとページに費やした時間
  • IP アドレス (匿名化)

お客様の情報の使用方法

私たちは収集した情報を次の目的で使用します。

  • お問い合わせやご要望にお答えします
  • Web サイトとサービスを改善する
  • 当社の Web サイトがどのように使用されているかを分析する
  • 私たちのウェブサイトのセキュリティと完全性を確保します

クッキー

当社では、Web サイトでのエクスペリエンスを向上させるために Cookie を使用しています。 Cookie は、お客様のデバイスに保存される小さなテキスト ファイルで、お客様によるサイトの使用状況を把握するのに役立ちます。

私たちは次の種類の Cookie を使用します:

  • 必須の Cookie:Web サイトが適切に機能するために必要です
  • 分析 Cookie:訪問者が当社の Web サイトをどのように操作するかを理解するためにご協力ください (同意がある場合のみ)

ブラウザの設定を通じて Cookie を制御できます。 Cookie を無効にすると、当社の Web サイトの機能に影響が出る可能性があることに注意してください。

サードパーティのサービス

当社は、Web サイトのトラフィック分析に役立つ Google Analytics などのサードパーティ サービスを使用する場合があります。これらのサービスは、お客様による当社ウェブサイトの使用に関する情報を収集する場合があります。

これらのサードパーティは独自のプライバシー ポリシーを持っているため、それらを確認することをお勧めします。

データセキュリティ

当社は、お客様の個人情報を不正なアクセス、改ざん、開示、または破壊から保護するために、適切な技術的および組織的対策を実施します。

あなたの権利

あなたには次の権利があります:

  • 自分の個人情報にアクセスする
  • 不正確な情報を修正します
  • 自分の情報の削除をリクエスト
  • データ処理の同意を撤回
  • あなたの情報の処理に反対します

これらの権利を行使するには、で提供される情報を使用して当社までご連絡ください。お問い合わせページ。

このポリシーの変更

当社は、このプライバシー ポリシーを随時更新することがあります。変更があった場合は、更新された「最終更新日」を付けてこのページに新しいポリシーを掲載することでお知らせします。

質問がありますか?

このプライバシー ポリシーについてご質問がある場合は、までお問い合わせください。お問い合わせください。
